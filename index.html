Indra Agent-Based Modeling System

Project Goal

Indra is an agent-based modeling system in Python. The ultimate goal is to build a GUI front-end that will allow non-coders to build models, while at the same time permitting coders to use Python for more flexibility in model creation.

Project Architecture

Overview of Ideas

Every model has two basic objects: an environment and an agent. Everything in every model is a node in a graph. Thus everything is interconnected through Indra's Net.

node.py

The Node class is the most abstract class of the Indra modules; or at least, “the core of the system.” From it, all other classes are derived. Everything is a node, making everything relatable under a common general structure.

Attributes

  • name - the name of the node
  • graph - initially None, may become a graph object. Thus every node may itself be a graph.
  • ntype - initially, the name of the class of node, often custom named (e.g. "agent," "sheep," "grid env," etc.)
